Privacy Policy – Đọc online

Last updated: 2025-12-05

Đọc online (“the Plugin”) is a KOReader plugin that allows users to access their Google Drive files through Google OAuth 2.0 authentication. This Privacy Policy explains how Google user data is accessed, used, protected, and handled.

1. Data We Access

The Plugin only accesses the minimum data required for its functionality, including:

  • Basic Google account information (such as email address) for authentication purposes.
  • Google Drive file metadata and file content when explicitly requested by the user.
  • OAuth 2.0 access tokens and refresh tokens provided by Google.

2. Purpose of Data Usage

Google user data is used solely to:

  • Authenticate users via Google OAuth 2.0.
  • Display the list of Google Drive files selected by the user.
  • Download files to the user’s device upon user request.
  • Maintain login sessions using OAuth tokens.

The Plugin does not modify, delete, or upload data to Google Drive.

3. Data Storage and Security

  • OAuth access tokens and refresh tokens are stored locally on the user’s device running KOReader.
  • The Plugin does not store or process Google user data on any developer-owned servers.
  • All communications with Google services are conducted over HTTPS and comply with OAuth 2.0 security standards.
  • Google user data is protected using industry-standard security measures, including encryption in transit and restricted access.

4. Data Sharing and Disclosure

  • The Plugin does not sell, rent, or share Google user data with any third parties.
  • Google user data is used exclusively to provide the Plugin’s core features.
  • Data may only be disclosed if required by applicable laws, regulations, or legal requests.

5. Data Retention and Deletion

  • Google user data is retained only for as long as necessary to provide the requested service.
  • Users can delete all locally stored tokens by logging out of the Plugin.
  • When access is revoked or the user logs out, all OAuth tokens are permanently removed from the device.

6. User Control and Rights

  • Users can revoke the Plugin’s access at any time via:
    Google Account → Security → Third-party access.
  • Users have full control over which files are accessed by the Plugin.
